# 📊 Dashboard RH – Power BI

## Présentation

Ce projet présente un tableau de bord interactif développé avec Power BI permettant de suivre les principaux indicateurs des Ressources Humaines.

L'objectif est d'aider les équipes RH à analyser les effectifs, les départs et la répartition des employés grâce à des visualisations interactives.

---

## Fonctionnalités

Le tableau de bord permet notamment de visualiser :

- Effectif total
- Nombre d'employés actifs
- Nombre de départs
- Taux de départ
- Âge moyen
- Ancienneté moyenne
- Répartition par département
- Répartition par tranche d'âge
- Répartition par genre
- Analyse des départs selon la tranche salariale
- Analyse des départs selon l'expérience

---

## Technologies utilisées

- Power BI Desktop
- Power Query
- DAX
- CSV

---

## Source des données

Les données utilisées proviennent d'un jeu de données public utilisé dans un tutoriel YouTube.

Ce projet a été réalisé dans un objectif d'apprentissage et de démonstration de compétences sur Power BI.
